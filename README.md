# IHK IT Specialist SI – Practice Pool

This is a practice pool for aspiring IHK IT Specialists in System Integration, covering examination parts AP1 and AP2. The project offers an interactive way to test and consolidate knowledge in various subject areas.

## Features

*   **Comprehensive Question Catalog**: Practice questions on relevant topics for AP1 and AP2.
*   **Filter Options**: Filter questions by categories or difficulty levels for targeted learning.
*   **Progress Display**: Track your learning progress and see how well you perform in individual areas.
*   **Answer Explanations**: Get detailed explanations for the correct answers to deepen your understanding.
*   **Light/Dark Theme**: Switch between a light and a dark theme to optimize readability.
*   **Responsive Design**: Optimized for use on various devices (desktop, tablet, smartphone).

## Technologies

*   **HTML5**: Structure of the web application.
*   **CSS3**: Styling and layout, including variables for theme switching.
*   **JavaScript (ES6+)**: Interactive logic, question management, and theme toggling.

## Installation

To run the project locally, follow these steps:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/PlanBrilliant/fisi-quiz.git
    cd fisi-quiz
    ```
    *(Note: Replace `https://github.com/PlanBrilliant/fisi-quiz.git` with the actual link to your repository if it's public.)*

2.  **Open in Browser**:
    Open the `index.html` file directly in your web browser. Alternatively, you can use a local web server (e.g., Live Server Extension for VS Code or `python -m http.server` in the project directory) to host the application.

## Usage

1.  **Filter Questions**: Select the desired categories or filter options to customize the questions.
2.  **Start Quiz**: Click the "Start" button to begin a new round of questions.
3.  **Answer**: Choose your answer from the given options or enter it into the input field.
4.  **Explanations**: After answering a question, the correct solution and an explanation will be displayed.
5.  **Switch Theme**: Use the toggle switch in the top right to switch between light and dark themes.
